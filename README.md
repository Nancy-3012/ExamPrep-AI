## Problem Statement
Students often struggle to prepare exam-oriented questions from large syllabus documents and notes. Traditional study methods are time-consuming and inefficient.

## Solution
ExamPrep AI uses Large Language Models combined with Retrieval-Augmented Generation (RAG) to generate syllabus-based exam questions such as MCQs, short answers, long answers, and viva questions directly from uploaded academic materials.

## Technologies Used
- Python
- Large Language Models (LLMs)
- Retrieval-Augmented Generation (RAG)
- Vector Databases
- Streamlit (UI)
